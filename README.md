# SLSTM (Covid-lstm-hosp)

Python code for predicting COVID-19 hospitalization at the state-level in the US using a parallel Spatial-LSTM.

Package requirements:
```
    tensorflow (tf-gpu)
    pandas
    numpy
    scikit-learn 
    keras
```
